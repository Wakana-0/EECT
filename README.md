## 安装 & 使用

#### 确认系统需求
首先，确认您的设备是否满足以下需求:

> [!NOTE]
> **兼容性：** 现支持 Windows 10，对Windwos 8的兼容性需要验证。

推荐配置
| **🖥操作系统** | **🗄运行内存** | **运行环境（Releases）** |
| :-----: | :-----: | :------: |
| **Windows 10** 及以上 | ≥4GB | *64位操作系统，基于x64的处理器* |

若已**满足要求**，可以进行接下来的步骤。

#### 下载 EECT
您可在此应用 GitHub 仓库中下载最新的 EECT。  
[转到releases页面下载最新正式版EECT](https://github.com/EECT/EECT/releases)
